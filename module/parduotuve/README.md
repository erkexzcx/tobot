# Parduotuve module

This module allows to buy/sell items in main shop.

## Example

```yaml
  - _module: parduotuve
    action: pirkti
    item: MA1
    amount: max
```
```yaml
  - _module: parduotuve
    action: pirkti
    item: MA1
    amount: 5
```
```yaml
  - _module: parduotuve
    action: parduoti
    item: MA1
    amount: max
```
